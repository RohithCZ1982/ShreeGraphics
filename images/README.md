## Note:
The website will automatically generate placeholder images if actual images are not available. 
Replace these placeholder images with actual photos of your printing work for the best results.

## Image Guidelines:
- Use high-quality images (minimum 1200px width for hero image)
- Optimize images for web (compress to reduce file size)
- Use JPG format for photographs
- Maintain consistent aspect ratios within each section
